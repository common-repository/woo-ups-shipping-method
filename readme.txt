UPS shipping methods for WooCommerce. Provide live shipping rates by UPS.

== Introduction ==
Manage shipping rates in woocommerce store.

== Description ==
It will provide the live shipping rates from UPS.

= Features =
 * Automatic UPS shipping cost calculation
 * All supported currencies by UPS
 * Per Item Packaging
 * Option to provide Negotiated rates by UPS
 * Debug Mode

== Installation ==

Follow the below instructions to install the plugin on your website: 
1. Download and unzip the latest release zip file.
2. Upload the entire plugin directory to your /wp-content/plugins/ directory.
3. Activate the plugin through the Plugins menu in WordPress Administration.
